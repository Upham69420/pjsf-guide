// ABOUT

What is PJSF?

The Philippine Joint Strike Force (PJSF) is the largest semi-milsim unit in the Philippines, combining realistic combined-arms tactics with a casual, welcoming community. We play Arma 3 — a hardcore military simulator — but without the rigid hierarchy that makes other units feel like a second job.

We run main operations every Friday and Saturday, 9PM–12AM PST. Open Deployments (Liberation, Hearts & Minds, Antistasi) run throughout the week on public servers open to non-members too.

No Strict RP

No "Yes Sir / No Sir." No attendance requirements. No deadlines. Life comes first. Our ranks represent individual achievement — not entitlement to special treatment.

What Scenarios Do We Play?

Primarily modern and contemporary warfare, but we've run everything: Black Hawk Down, Star Wars, Vietnam, WW2, Fallout, Ukraine, Alien, Cowboys, Warhammer 40K, Half-Life, Zombie, Cowboy, Pirate, Bosnia, Iraq War, Mindanao, and a lot more. If it's in Arma, we've probably done it.

// ELIGIBILITY

Requirements

Before submitting an application, make sure you meet all of the following:

  • You are 16 years old or above
  • You own a legitimate, purchased copy of ArmA 3 on Steam — cracked versions are not allowed
  • You have read the PJSF Community Constitution & Charter
  • You are willing to download and install TeamSpeak v3.6.0 for operations comms
DLC Requirements

No DLC is required. SOG: Prairie Fire is recommended if you want to join Vietnam-era operations.

Mobile / Android?

No. Arma 3 requires a Windows PC. It is not available on mobile or console.

// GAME SETUP

Mod & TeamSpeak Setup

Part 1 — Mods

1

Download the Modlist File

Get the latest Base Mods V15.1 modlist file from the #mod-updates channel in Discord (pinned message — 31 mods). This is an HTML file you import into the Arma 3 Launcher.

2

Import into Arma 3 Launcher

Open Arma 3 Launcher → Mods tab → click the ≡ icon → Import. Select the modlist file you downloaded. The launcher downloads all mods from the Steam Workshop automatically. ▶ Watch the video guide

About the ~60GB Size

Mods are large because Arma 3 is set in 2035 by default. Our base mods reset the game to a contemporary era with real-world vehicles and weapons. The list is intentionally kept small and optimized. You can delete non-base mods after each operation to save disk space.

Part 2 — TeamSpeak 3

1

Download & Install TeamSpeak 3.6.0

Download TeamSpeak 3.6.0 here (Windows 64-bit). TeamSpeak simulates a military radio environment integrated directly with Arma 3. Required for all operations.

2

Set Up TFAR Integration

TFAR (Task Force Arrowhead Radio) links Arma 3 with TeamSpeak. Follow the step-by-step setup guide. Confirm the TFAR plugin shows as Enabled in TeamSpeak's Plugins menu.

3

Connect to the PJSF TeamSpeak Server

Address: ts.arma-ph.net or 15.235.199.115 (standard port). ▶ Video: How to join the TS server

4

Set Up Push-to-Talk

Configure a Push-to-Talk (PTT) key in your TeamSpeak settings. Required for in-game communication. Ask in #staff-help-desk if you need help.

// UNITS

Elements & Units

PJSF is organized into specialized branches called Elements. Each element has its own role, training pipeline, and command structure. Most members eventually specialize by becoming billeted — an official assignment to an element. It's optional, but unlocks exclusive roles, training, and operations.

Billeted vs. Unbilleted

Billeted members are the core of their element — they have priority for exclusive ops, private comms, and greater responsibility. You can only hold one billet at a time, but transfers are possible. Some roles require qualifications even without a billet.

// FAQ

Frequently Asked Questions

Do I have to roleplay? Are you guys strict?
No. PJSF is not a hardcore unit. No attendance requirements, no "Yes Sir/No Sir" culture, and no deadlines. A baseline of respect and discipline during gameplay is expected, but life always comes first.
When do operations happen?
Main operations run Friday and Saturday, 9:00 PM – 12:00 AM Philippine Standard Time (PST). Open Deployments run Sunday–Thursday, time varies. Training schedules depend on instructor availability.
How long does my application take?
Usually reviewed within the day, but can take up to 1–3 days since each application is manually reviewed. Orientation scheduling can take an additional 1–3 days. You can join open deployments as soon as your application is approved.
What DLC do I need?
None are required. SOG: Prairie Fire is recommended for Vietnam-era operations, but not mandatory.
How big are the mods?
About 60GB for the base modlist. Mods replace Arma's default 2035 setting with a modern, contemporary era using real-world vehicles and weapons. You can delete non-base mods after each operation if disk space is tight.
What roles can a new member slot in?
Unqualified members can slot in as Rifleman, Automatic Rifleman, Anti-Tank Rifleman, or Grenadier in Dagger (infantry). Team Leader slots may be available at Squad Leader discretion. Other roles require rank or specialized training.
Can I join more than one element?
No. You can only hold one billet at a time. Transfers are allowed with element lead approval. Choosing not to join an element is also valid, though some roles and occasional ops may be locked.
Do I have to attend briefings?
Strongly recommended, but not mandatory. If you miss it, read the OPORD on the website and ask your Team Leader to fill you in in-game. Win conditions, assets, and special rules differ every operation.
Can I play on mobile?
No. Arma 3 requires a Windows PC. It is not available on mobile or console.
